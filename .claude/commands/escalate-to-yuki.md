---
description: 로컬 세션(유키 COO)에서만 가능한 작업을 `유키` 라벨 이슈로 에스컬레이션
---
이 레포에서 클라우드 세션(원격 에이전트)이 해결할 수 없는, **본부 로컬 세션에서만 가능한
작업**이 필요할 때 사용한다.

수행 절차:
1. 필요한 로컬 작업을 정리한다 — 무엇을 / 왜 / 기대 결과. (인자: $ARGUMENTS)
2. 다음을 실행해 이슈를 생성한다:
   `gh issue create --label "유키" --title "[유키-LOCAL] <한줄요약>" --body "<무엇을/왜/기대결과/관련파일>"`
3. 생성된 이슈 URL 을 사용자에게 보고한다.

유키(COO)가 본부 로컬 세션 시작 시 `유키` 라벨 monitor 로 감지 → 처리 후 이 이슈에 회신/close.
