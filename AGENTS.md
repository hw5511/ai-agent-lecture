# AGENTS.md — ai-agent-lecture

## 레포 목적
AI 에이전트 강의 랜딩페이지.

## 유키 에스컬레이션 규약 (필수)

이 레포는 클라우드 세션(claude.ai / Codex 등 원격 에이전트)이 작업할 수 있다.
**클라우드 세션에서 해결 불가능한 일 — 본부 로컬 세션에서만 가능한 작업**이 필요하면
직접 시도해서 막히지 말고, 유키(COO)에게 에스컬레이션한다.

로컬 전용 작업 예시:
- 본부 PC 파일시스템 / 로컬 폴더 접근
- PM2 프로세스 관리 (start/stop/배포)
- 로컬 배포·빌드 / 로컬 서버 기동
- keyring·시크릿·인증 토큰
- 로컬 전용 CLI (wi.py 등) / 본부 네트워크 (Cloudflare WARP 등) / 하드웨어

**에스컬레이션 방법**:
1. 이 레포에 GitHub 이슈 또는 PR 을 생성한다.
2. **label = `유키`** (필수 — 이 라벨이 있어야 유키 monitor 가 감지한다).
3. 제목 접두사 `[유키-LOCAL]`.
4. 본문에 필요한 로컬 작업을 구체적으로 적는다 — 무엇을 / 왜 / 기대 결과.

유키는 세션 시작 시 `유키` 라벨 이슈/PR 을 감지하는 monitor 를 상시 가동한다.
감지되면 유키가 본부 로컬에서 처리한 뒤 해당 이슈/PR 에 회신하거나 머지한다.
